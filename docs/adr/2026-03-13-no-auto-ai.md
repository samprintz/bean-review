# No Automatic AI Account Prediction

`bean-ai` predicts accounts for Beancount transactions
based on their narration.
It was trained on transactions with clean, human-readable narrations.

Transactions from `ledger-telegram-bot`
already carry proper narrations at import time,
so `bean-ai` could in principle be invoked automatically
during import for those.

However, transactions from bank accounts and other sources
arrive with raw, unreviewed narrations
that are not yet suitable for prediction.

Applying `bean-ai` automatically in a post-import hook for all sources
would therefore yield poor predictions for most transactions.

## Decision

`bean-ai` is not applied automatically.
It is invoked manually by the user
— after the narration has been reviewed and edited.

## Consequences

- Predictions are only made when the narration is clean,
  keeping accuracy high.
- The workflow requires the user to edit the narration
  before requesting a prediction.
